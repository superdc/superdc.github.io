##一维数组##

一维数组的声明
datatype arrayName[size]
类型说明符（int，char，float。。。） 数组名[常量表达式：数组大小]
数组大小：

  1. #define N 20 int a[N];
    第一种方式是可以的，define定义的常量由编译器放在常量区，内存中不存在，没有地址，无从修改。

  2. const int num = 10; int a[num];
    第二种方式是不可以的，因为const定义的是可变的常量。const限制不能由程序修改，但在内存中存在实体，可以在内存中修改。

    初始化
    类型 数组名[宽度] = {初值列表}；
    初始化的元素少于数组长度时，剩余的元素均初始化为0；
    直接初始化相当于对每一个元素赋值，会自动进行数据类型转换；
    数组元素的多少可以确定的时候，数组的大小可以省略；
    int a[]={};是不可以的，不能确定元素的个数，分配合适的内存。

二维数组的引用

  • 数组名[行下标表达式][列下标表达式]
  • 只能逐个引用各行各列的元素，不能整体引用
  • 行下标和列下标均不作越界检查
  • a[i]代表第i-1行的首地址
  • &a[i][j]等价于a[i]+j; a[i][j]等价于*(&(a[i][j]))
  • a[i][j]只能代表元素，与*(&a[i][j])等价，&a[i][j]和a[i]+j可以代表a[i][j]
