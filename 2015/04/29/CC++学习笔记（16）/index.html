C/C++ 学习笔记(Windows)

##字符串##
C语言中没有提供字符串这个特定类型，而是以特殊字符数组的形式来存储和处理字符串，这种空字符数组必须以空字符’\0’结尾。因此，也将这种特定字符数组称为C风格字符串。

###字符串与字符数组的关系###

  • 字符串用一维字符数组存放
  • 字符数组具有一维数组的所有特点：
  • 数组名是指向数组首地址的地址常量
  • 数组元素的引用方法可用指针法和下标法
  • 数组名做函数参数是地址传递,占4个字节
  • 输入输出方式：%s, %c

###字符指针变量和字符串数组###

  • 字符数组的每个元素中存放一个字符；字符指针变量中存放的是地址（字符串第一个字符的地址）
  • 可以对字符指针变量赋值，不能对数组名赋值，数组名是常量
  • 编译时为字符数组分配若干存储单元以存放各元素的值；字符指针变量，只分配一个存储单元存储地址

###输入输出###

字符数组可以整体输入输出，不可以整体操作，如赋值，加减乘除

###字符串查找函数strstr()###
函数原型：char strstr(const char str1, const char str2);
功能：找出str2字符串在str1字符串中第一次出现的位置（不包括str2的串结束符）。若找到，返回改位置指针；否则，返回空指针。

###字符串比较函数strcmp()###
函数原型：int strcmp(const char str1, const char str2)
功能：比较两个字符串的大小(区分大小写)。
返回值说明：如果str1比str2小，返回负值；如果str1，str2相等，返回0；如果str1大于str2，返回正值。
说明：字符串的比较是比较字符串中各对字符的ASCII码。首先比较两个串的第一个字符，若不相等，则停止比较并得出大于或小于的结果；如果相等就接着比较第二个字符然后第三个字符等等。
① 字符串1小于字符串2,strcmp函数返回一个负值;
② 字符串1等于字符串2,strcmp函数返回零;
③ 字符串1大于字符串2,strcmp函数返回一个正值
mystrcmp()

windows的文件夹按文件名称排序是不区分大小写的，实现windows排序需要使用strupr()函数将小写字母转为大写字母

###字符串比较函数strncmp()###
函数原型：int strncmp(const char str1, const char str2,int len)
功能：比较字符串前len个字符的大小。返回值与strcmp()相同。

###字符查找函数strchr()###
函数原型：char strchr(const char str, char c)
功能：查找字符串str中首次出现出现字符c的位置.（可以从str的任意位置开始查找）
返回值：返回首次出现字符c的位置，如果c不在s中，则返回NULL。

###字符串复制函数strcpy()###
函数原型：char strcpy(char dest, const char *src);
功能：把从src地址开始的字符串复制到以dest为地址开始的内存空间。src和dest所指内存区域不可以重叠且dest必须有足够的空间容纳src字符串。
返回值：dest的地址

###字符串长度函数strlen()###
函数原型：unsigned int strlen(const char *s)
功能：计算字符串s的长度，不包括’\0’在内
返回值：返回字符串长度。

###字符串拼接函数strcat()###
函数原型：char strcat(char dest, const *source);
功能：把source所指字符串添加到dest结尾处（覆盖dest结尾处的’\0’）并添加’\0’.
返回值：返回指向dest的指针。

###字符串拼接函数strncat()###
函数原型：char strncat(char dest, char *source, int n);
功能：把source所指的字符串的前n个字符添加到dest的结尾处（覆盖dest结尾处的’\0’），并添加’\0’

###字符串转整数atoi函数###
功能：将数字字符串转换为整数。转换成功，返回整数；转换失败，返回0；
注意：如果有非数字字符出现，就会转换失败，返回0.

###字符串设置函数strset()###
函数原型：char strset(char str, char c)
功能：将一个字符串中的所有字符都设为指定字符

###字符串倒转函数strrev()###
函数原型：char strrev(char str);
功能：串倒转
char mystrrev(char str)
{
if(str==NULL)
{
return NULL;
}
int length = strlen(str);
for (int i = 0; i < (length / 2); i++)
{
char ch = str[i];
str[i] = str[length - 1 - i];
str[length - 1 - i] = ch;
}
return str;
}

###字符串小写转大写函数strupr()###
函数原型：extern char strupr(char s)
功能：将字符串s中的小写字母转换为大写字母，其他字符不变

###字符串大写转小写函数strlwr()###
函数原型：extern char strlwr(char s)
功能：将字符串s中的大写字母转换为小写字母，其他字符不变

###atoi函数###
写atoi函数的时候需要注意一下几点

  1. 字符串前的空白

  2. 字符串所表示数值的正负号

  3. 结束条件，遇到非数字或者字符’\0’结束

  4. 考虑溢出，分别与int值所能表示的最大(0x7fffffff)和最小值(0x8000000)进行比较

  5. 考虑异常输入情况下，用全局变量valid来标识，对于”+/-“ “0” “+abc”需要进行区分
