#Linux C 一站式学习(Ubuntu)#

##第一个程序##

保存为hello.c，打开终端编译运行：

  1. gcc hello.c
  2. ./a.out（中间没有空格）
  3. 打印出了hello world

说明：

  • 若不指定可执行文件名，默认生成a.out文件。可通过指令gcc hello.c -o hello指定可执行文件名为hello，运行./hello即可。
  • C程序生成可执行文件：编译器翻译成汇编程序->汇编器翻译成机器指令->链接器处理->可执行文件
  • gcc -Wall hello.c 可显示警告信息
  • 注释：/*/可进行跨行注释，并且可以穿插在一行代码中，如printf(/comment*/“hello world\n”); 但//注释不可以跨行，不可以穿插在一行代码之中
  • 格式化字符串：%+转换说明，是在运行时由printf函数处理的

printf("character: %c\n integer: %d\n floating point: %f\n",'}',34,3.14);

##变量##
变量的类型和名字由声明给出。
声明和定义：分配存储空间的声明同时也是定义，不分配存储空间的声明不是定义。
声明有三种类型
变量声明，函数声明，类型声明（类型声明总是不分配存储空间的，严格说来只有类型说明而没有类型定义）。

##关于整数除法##
两个整数相除，结果仍为整型。除不尽的情况下，小数部分直接省略。正数情况下，相当于向下取整；负数的情况下，相当于向上取整。
练习：假设变量x和n是两个正整数，我们知道x/n这个表达式的结果要取Floor，例如x是17，n是4，则结果是4。如果希望结果取Ceiling（例如x是17，n是4，则结果是5；x是16，n是4，则结果是4），应该怎么写表达式呢？
A.（x+n-1）/ n可实现向上取整
B. (int)((float)x/(float)n)+0.9)可实现向上取整

##关于头文件##
使用math.h中声明的库函数，gcc命令行必须加 -lm 选项，因为数学函数位于libm.so库文件中，-lm告诉编译器程序中用到的数学函数要到limb.so库文件中找。
其他大部分库函数位于libc.so库文件中，使用libx.so中的库函数在编译时不需要加 -lc 选项，因为这个选项是gcc的默认选项。

##$?##
$?是Shell中的一个特殊变量，表示上一条命令的推出状态。

##关于main函数##

  1. main(){…} 不写返回值类型，不写参数列表，属于Old Style C的风格–>不建议使用
    不写返回值类型，表示返回int型；不写参数列表，表示参数类型和个数没有明确指出。
  2. 操作系统调用main函数时是传参数的，main函数的标准形式是 int main(int argc, char *argv[]);C标准也允许int main(void)这种写法。

##函数返回值##
函数返回一个值相当于定义一个和返回值类型相同的临时变量并用return后面的表达式来初始化。而且这个临时变量不在内存中，而是在寄存器中，所以不能不为左值。

函数练习
编写一个布尔函数int is_leap_year(int year)，判断参数year是不是闰年。如果某年份能被4整除，但不能被100整除，那么这一年就是闰年，此外，能被400整除的年份也是闰年。

编写一个函数double myround(double x)，输入一个小数，将它四舍五入。例如myround(-3.51)的值是-4.0，myround(4.49)的值是4.0。可以调用math.h中的库函数ceil和floor实现这个函数。
