C/C++ 学习笔记(Windows)

##运算符与表达式（一些概念）##

  • 参与运算的对象是操作数。
  • 运算符是指定要执行某项操作的一个标志符号。指明了对操作数进行的运算。
  • 表达式是一个求值的规则。由变量，常量，运算符，函数和圆括号按一定的规则组合而成。
  • 表达式有算术表达式，关系表达式和函数表达式。
  • 表达式书写规则：
  • 运算符不能相邻
  • 乘号不能省略
  • 可以用多层括号表示运算次序，括号必须成对出现，都是小括号
  • C语言有32个关键字，34个操作符，9种控制语句

###算术运算符###
+ - * / %

  • 除：当除号两边都是整型量时，结果只保留整数部分，小数部分被省略；当有一边为实数类型时，结果是实数。

###自增自减运算符###

  • num++ <=> num = num + ;**
  • num– <=> num = num - 1;**
  • 注意：此运算符只能对变量使用，不能用于常量和表达式**
  • 做前缀时，先运算，后引用**
  • 做后缀时，先引用，后运算**
  • ++/–的优先级高于加减乘除法**
  • a=3;b=5; c=(a++)b => a=4;c=15*
  • ++/–的结合性是从右向左

++的应用

###赋值运算符###

  • += -= *= /= %= <<= >>= &= ^= |=
  • 含义：exp1 op= exp2 <=> exp1 = exp1 op exp2
  • 注意：每个运算符不能分开写
  • 功能：将数据（常量，变量或表达式）赋给一个变量
  • 赋值转换规则：赋值号右边表达式的值自动转换成其左边变量类型的值
  • 赋值运算符从右向左开始执行
  • 赋值表达式的值就是最终变量的值

###逗号运算符（顺序求值运算符）###

  • 用逗号将多个表达式连接起来，逗号运算符从左向右执行，整个表达式的值是最后那个逗号之后表达式的值
  • 逗号运算符优先级最低，任何其他运算符的优先级都比逗号运算符优先级高
  • 逗号用于求值时是顺序求值运算符，否则作为间隔，可以用于连接多个语句，printf("hello"),printf("world");输出helloworld

###关系运算符###

  • < > <= >= == != (最后两个优先级低)**
  • 形式：表达式 运算符 表达式
  • 两个表达式可以是整型，浮点型和字符型。整型和浮点型比较的是打小，字符型比较的是ASCII码值
  • 关系表达式成立，表达式的值为1，不成立，表达式的值为0
  • 多个同等优先级的关系运算符，是从左向右执行
  • 1<2<2 结果为1 首先，1<2，成立，值为1，然后，1<2，成立，值为1，所以表达式值为1

##左值，右值和程序实体##

  • 程序实体是内存里面可以标识的一片内存区域，可以针对内存区域进行任意操作。
  • 左值是左值表达式的简称，是指明一个程序实体的表达式。
  • 能放在赋值号左边的表达式是左值，它指明了一块内存区域，赋值运算是改变这一区域的操作。
  • 左值并非一定可以放在赋值号左边，const唱两首左值，但不能将其放在赋值号左边。
  • 表达式a-1不是左值，在内存中找不到，是在cpu寄存器中完成的
  • 右值是在赋值号右边取出值赋给其他变量的值，可以在内存中，可以在cpu中
