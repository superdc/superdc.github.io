C/C++ 学习笔记(Windows)

##位运算##

###位运算符###
位逻辑运算符：&（位与），^（位异或），|（位或），~（位取反）
移位运算符：<<（左移），>>（右移）
注意：

  • 取反运算符不改变变量原来的值，通过赋值运算符改变
  • 位与运算：与0相与实现按位清零；与1相与实现按位保留实现取出某些位的功能
  • 位或运算：与0相或保持不变；与1相或都变为1
  • 位异或运算：相同为0，不同为1.与0异或，不变；与1异或，取反。

技巧

  • 位异或运算可以不借助中间变量实现两个数据的交换：x=x^y; y=y^x; x=x^y;不会溢出(或者x=x+y; y=x-y; x=x-y;有可能溢出。或者x=x*y;y=x/y;x=x/y;有可能溢出)
  • x&~1可以实现x末尾清零，不用管x有几位
  • 可以用与运算符代替求余运算符（当x是2的幂次的时候成立）： a&(x-1);比求余运算符ba%b=a-(a/b)*b;速度快
  • 移位操作在CPU寄存器中完成；左移一位的值相当于原来的值乘以2，要注意溢出问题；右移一位的值相当于原来的值除乘以2；

###位运算复合运算符###

  • a<<=2 -> a=a<<2
  • a>>=2 -> a=a>>2
  • a&=2 -> a=a&2
  • a|=2 -> a=a|2
  • a^=2 -> a=a^2
  • a=~a ~只有一个操作数，没有复合运算符

###位运算注意事项###

  • 位运算只适用于整数和字符，不适用于实数
  • 位运算位数不匹配的时候，自动进行数据类型转换（如char类型与int类型进行位运算，char类型自动转换为int类型）。无符号整数或有符号整数，自动填充0；有符号负数自动填充1.都是按符号位填充。
  • 位左移，右边填充0，不区分有无符号，正负数；位右移，按符号位填充，即负数左边填充1.
  • 位反，位与，位或和位异或运算符不关心操作数的符号，只是按照操作数所在字符的0、1序列进行比对，符号位会被当做普通的0和1进行处理.
  • x=x&(x-1)相当于求x中含有1的位数（相与一次减少一个1）

###位字段###
位字段限定结构体某个变量用固定的二进制位，节约内存
struct date
{
unsigned int day : 5;//限定用5位二进制表示，越界会溢出，因字节对齐填充的位不能使用
unsigned int month : 4;//位字段不能设置为0，要大于0
unsigned int year : 14;
};//占4个字节，不使用位字段占12个字节
不可以取位域的地址。位域通过位运算来实现，位域变量成员的地址不能
如果结构体内两个字符的位数少于8位，会填充为一个字节，通过位操作操作位字段
使用位字段仍然有结构体字节对齐的问题

###打印补码###

打印浮点数二进制数
与打印整数的二进制数不同，浮点数不能直接利用位运算进行操作，因为位运算只适用于整数和字符。
所以利用浮点数的地址取浮点数在内存中的二进制数。
数据在内存中的存储：高字节在后，低字节在前。
对于一个字节，高位在前，低位在后
